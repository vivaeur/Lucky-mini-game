 Présentation du mode de jeu




                    
         Lucky Tower est un mode de jeu inspiré de l’idée des Lucky Blocks, combinant chance, défis et exploration verticale.  Ce mode offre une expérience unique et pleine de surprises
         surprises, où les joueurs doivent descendre dans une tour en bravant des épreuves aléatoires générées par des lucky blocks. Voici un aperçu détaillé :
   
           Dans Lucky Tower, vous progressez étage par étage dans une tour composée de niveaux générés de manière procédurale. Chaque étage contient :

           Un ou plusieurs blocs chanceux (Lucky Blocks) : Lorsque vous cassez ces blocs, des événements aléatoires se produisent. Ils peuvent être :
            
           Bénéfiques : Objets rares, enchantements puissants, alliés temporaires.
           Néfastes : Pièges, monstres, explosions, malédictions.
           Neutres : Modifications de l’environnement, puzzles à résoudre.
